# Meta Cryptovoxels

Third party client to render live Cryptovoxels parcels in three.js. You can hover over parcels on the map in order to load them in.

## Install

```
$ git clone https://github.com/webaverse/cv.git
$ cd cv
$ npm install
$ node index
```
